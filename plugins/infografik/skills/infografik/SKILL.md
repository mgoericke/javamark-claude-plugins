# Infographic Skill

## Purpose
Create professional infographics that visually present data and processes.

## Technology
- **matplotlib** for charts and diagrams
- **PIL/Pillow** for image composition
- **Output formats**: PNG (default), PDF (for print)

## Design Principles
- Clear color palette (max. 5 primary colors)
- Readable font sizes (min. 12pt)
- Maximize data-to-ink ratio
- Consistent icons and symbols
- Accessible color combinations

## Use Cases
- Process overviews (e.g. order processing)
- Statistics dashboards
- Comparison charts
- Architecture overviews
- Timelines and roadmaps
